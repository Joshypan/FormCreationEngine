Help Page

Login Help

To login, simply put your username and password into the boxes and click login. If you encounter any issues with the login process, please ensure that your username is correct and your password meets the required criteria.

File Upload

To upload a file, select a .txt or .csv file with the appropriate format. The file should contain survey questions with specified formats. After uploading, you can take your survey and view the results. If you face any issues with file upload, ensure the file format is correct.

The appropriate format is mentioned below:
Question Number,Question Type,Question Name,Question Parameters
1,MC,What is your favorite class?,Math`Science`Gym`History
2,ER,Explain photosynthesis,N/A
3,CB,Do you workout?,Yes`No
4,D,When were you born?,N/A
5,T,At what time do you wake up?,N/A
Note: " ` " is used to separate multiple values

Account Creation

To create an account, enter a username and a password. For your security, make sure your password contains a mix of uppercase and lowercase letters, numbers, and special characters.

What Is The Purpose?

This web application allows you to either manually create a survey or upload a file to create the survey. This survey can be taken many times so you can get a wide variety of results. It is easy to use and fast! We look forward to meeting your survey needs.
